## IO contract

### Function

```js
const { calculateExposure } = require('./src/exposure.js');
const result = calculateExposure({ holdings });
```

### Input

```ts
type Holding = {
  symbol?: string;
  quantity: number;   // finite number, may be negative for short/net exposure rows
  price: number;      // finite number in the row currency
  sector?: string;    // defaults to "Unclassified"
  currency?: string;  // defaults to "UNKNOWN"
};

type Input = { holdings: Holding[] };
```

Bounds and validation:

- `holdings` must be an array.
- Maximum `holdings.length` is 1000.
- `quantity` and `price` must be finite JavaScript numbers.
- No expressions, callbacks, user code, network locations, file paths, credentials, or dynamic vocabulary are accepted.

### Output

```ts
type ExposureResult = {
  total_value: number;
  positions: Array<{ symbol: string; value: number; weight: number }>;
  exposure_by_sector: Record<string, number>;
  weights_by_sector: Record<string, number>;
  exposure_by_currency: Record<string, number>;
  weights_by_currency: Record<string, number>;
  largest_position: { symbol: string; value: number; weight: number } | null;
  position_count: number;
};
```

Numeric outputs are rounded to 12 decimal places for deterministic JSON comparison. Weight fields are zero when total net value is zero.
